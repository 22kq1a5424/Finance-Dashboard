# Finance-Dashboard

Zorvyn Fintech Dashboard
Overview

Zorvyn Fintech Dashboard is a responsive web-based financial dashboard designed to track income, expenses, and overall financial trends. It provides an interactive interface for users to manage transactions and visualize financial data using charts.

Features
Dashboard with balance, income, and expense summary
Interactive charts for financial analysis
Insights section with trend and category-based graphs
Transaction management (add, edit, delete)
Role-based access:
Viewer: Can only view data
Admin: Can manage transactions
Dark mode toggle
Responsive design for mobile, tablet, and desktop
Sidebar navigation with toggle functionality
Technologies Used
HTML
CSS
JavaScript
Chart.js
Project Structure
index.html – Main structure of the application
style (internal CSS) – Handles layout and responsiveness
script (internal JS) – Manages data, charts, and interactions
How to Run
Download or clone the repository
Open the index.html file in any web browser
Start interacting with the dashboard
Usage
Use the sidebar to navigate between sections
Switch between Viewer and Admin using the dropdown
Admin users can add, edit, and delete transactions
Toggle dark mode using the switch in the top right
View charts in Dashboard and Insights sections
Future Improvements
Add user authentication system
Store data using local storage or database
Add filters (weekly, monthly, yearly)
Improve UI with animations and advanced design
Convert into a React-based application
Conclusion

This project demonstrates frontend development skills including UI design, state management, responsiveness, and data visualization. It can be extended into a full-stack financial management system.
